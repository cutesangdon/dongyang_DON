CSS의 개요 및 기본 구조 학습

- CSS 삽입 방법 학습

  1. 내부 스타일 시트: HTML문서 내에서 스타일을 정의
  2. 외부 스타일 시트: 별도의 CSS파일을 생성하여 HTML 문서와 연결 후 사용 (현재 페이지에 적용)
  3. 인라인 스타일: HTML문서 내에 적용할 대상에 직접 표시 (현재 페이지에 일부 적용)

- CSS 선택자 학습

  1. 전체 선택자: CSS문서 내에 * 선택자를 사용하여 페이지 내에 모든 요소에 스타일을 적용
  2. 태그 선택자: HTML의 태그를 선택자로 사용하여 해당 태그에 스타일을 적용
  3. Class 선택자: 요소의 Class를 선택자로 사용하여 해당 태그에 스타일을 적용
  4. ID 선택자: 요소의 ID를 선택자로 사용하여 해당 태그에 스타일을 적용

- 캐스캐이딩의 의미와 원칙에 대해 학습

- 선택자 우선순위에 대해 학습

  1. !important 적용
  2. 인라인 스타일 적용

  4. 클래스 선택자

- 인라인 레벨 요소: 수평방향으로 디스플레이되며 줄을 차지 하지 않는 요소

- 블록 레벨 요소: 요소 삽입 시 한줄 씩 차지하며 수직 방향으로 디스플레이
